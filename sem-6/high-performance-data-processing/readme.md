# SECP3133 - High Performance Data Processing

## 📌 Course Overview
This course introduces the concepts of parallel and distributed computing for processing large-scale datasets efficiently. It focuses on improving computational performance using advanced programming techniques and architectures.

---

## 🎯 Learning Outcomes
- Understand parallel computing concepts
- Apply MPI and OpenMP techniques
- Analyze system performance optimization
- Develop scalable computing solutions

---

## 📚 Topics Covered
- Parallel computing models
- Distributed systems
- MPI programming
- OpenMP programming
- Performance optimization techniques

## 🛠 Skills Developed

- Low-level computing understanding

## 📖 Reflection
This course provided deep insight into how large-scale computing systems handle massive amounts of data efficiently through parallel processing techniques. Initially, understanding concepts such as thread synchronization, process communication, and distributed computing models was challenging due to their technical complexity. However, practical implementation using MPI and OpenMP helped me understand how tasks can be divided and executed simultaneously to improve performance.

This subject is particularly important in the field of Data Engineering, where processing large datasets efficiently is a core requirement. I learned how system performance can be optimized by reducing computation time through parallel execution. It also improved my understanding of hardware-level and software-level interactions in computing systems. Overall, this course enhanced my technical depth and prepared me for working with big data systems and high-performance computing environments in real-world industry applications.
