# SECR2213 - Network Communication

## 📌 Course Overview
This course introduces computer networking concepts including protocols, architecture, and data transmission systems.

## 🎯 Learning Outcomes
- Understand OSI and TCP/IP models
- Analyze network communication processes
- Identify network protocols

## 📚 Topics Covered
- OSI model
- TCP/IP protocol suite
- Routing and switching

## 🛠 Skills Developed
- Network analysis
- System communication understanding

## 📂 Coursework / Projects
- Network topology simulation
- Protocol analysis assignment

## 📖 Reflection
This subject helped me understand how data is transmitted across networks, which is essential in modern distributed systems. Learning about the OSI and TCP/IP models gave me a structured view of how communication between devices occurs. Although some networking concepts were initially abstract, practical simulations helped clarify how data packets move through networks. This course is important for understanding how applications interact over the internet, especially in cloud computing and distributed systems. It enhanced my appreciation of backend systems and network infrastructure.
