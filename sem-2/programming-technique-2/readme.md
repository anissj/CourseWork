This course extends programming knowledge using object-oriented programming concepts in C++.

- Design modular programs
- Use inheritance and polymorphism

- Inheritance
- Polymorphism

## 🛠 Skills Developed

- System design basics

## 📖 Reflection
This course expanded my understanding of programming by introducing object-oriented concepts. Unlike procedural programming, OOP allowed me to structure programs more efficiently and logically. Concepts such as inheritance and polymorphism initially required deeper understanding, but practical implementation helped reinforce learning. This subject improved my ability to design scalable and reusable code, which is essential in software development. It also prepared me for more advanced programming subjects in later semesters.
