# SECJ3553 - Artificial Intelligence

## 📌 Course Overview
This course provides an introduction to Artificial Intelligence (AI), focusing on the fundamental concepts, techniques, and algorithms used to develop intelligent systems. It covers both theoretical foundations and practical applications such as search algorithms, machine learning basics, and knowledge-based systems. The subject serves as an important bridge between traditional programming and modern data-driven intelligent systems.

---

## 🎯 Learning Outcomes
- Understand core concepts of artificial intelligence
- Apply search and optimization algorithms
- Develop basic machine learning understanding
- Analyze intelligent system design approaches

---

## 📚 Topics Covered
- Search algorithms (DFS, BFS, A*)
- Knowledge representation
- Machine learning fundamentals
- Problem-solving strategies
- Intelligent agents

## 🛠 Skills Developed

- Problem modeling

## 📖 Reflection
This course significantly enhanced my understanding of how intelligent systems are designed and how machines can simulate human-like decision-making processes. Initially, AI concepts such as heuristic search and optimization algorithms were challenging because they required abstract thinking and mathematical reasoning. However, through hands-on implementation of search algorithms like BFS, DFS, and A*, I was able to better understand how problems can be solved efficiently using structured approaches.

Furthermore, the introduction to machine learning provided me with insight into how data-driven models are trained and evaluated. This is particularly relevant to my specialization in Data Engineering, as AI systems heavily rely on high-quality data pipelines and structured datasets. Overall, this subject strengthened my analytical thinking, improved my programming skills in Python, and gave me a clearer understanding of how AI is applied in real-world systems such as recommendation engines, predictive analytics, and automation tools.
