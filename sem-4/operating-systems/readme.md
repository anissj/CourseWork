
# SECR2043 - Operating Systems

## 📌 Course Overview
This course covers the principles of operating systems including process management, memory management, and system scheduling.

## 📖 Reflection
This subject provided a deeper understanding of how operating systems manage hardware and software resources. I learned about processes, threads, scheduling algorithms, and memory management techniques. Initially, concepts such as deadlock and synchronization were difficult to visualize, but diagrams and simulations helped clarify them. This course is important because it explains what happens behind every running application. It also improved my understanding of system efficiency and resource allocation, which is essential for system-level programming and performance optimization.
